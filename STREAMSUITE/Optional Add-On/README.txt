STREAM SUITE — DIAPER CHECK [OPTIONAL ADD-ON]

Made by BabyDreamsy

This is an optional ABDL-themed add-on and is not part of the main Stream Suite import.

CONTENTS
- One Streamer.bot action: STREAM SUITE - Diaper Check
- One command: !check
- The command is disabled by default after import.

ACTION COMMENTS
- The action contains only the approved green summary comment and light-blue Feature Summary reminder.

DEPENDENCY
This add-on reads the SeenUsers global variable created by the main Stream Suite attendance system.
A target must already be recorded in SeenUsers before the check succeeds.

INSTALLATION
1. Install the Stream Suite main package first.
2. Import Stream_Suite_Diaper_Check.sb.
3. Review the command text and response phrases.
4. Enable !check only when the channel wants this feature.
5. Test a self-check and a targeted check with attendance data present.
