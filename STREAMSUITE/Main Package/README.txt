STREAM SUITE — MAIN PACKAGE

Made by BabyDreamsy

Stream Suite: Built for Streamers. Powered by Community.

This main package contains 25 Streamer.bot actions and 15 commands.
Diaper Check is intentionally distributed as a separate optional add-on.

ACTION COMMENTS
- Every action has exactly two standard comments: a green action summary and a light-blue Feature Summary reminder.
- Reset actions also include 2 additional comments: a light-purple reminder for optional use of a 
Stream Deck Button and a orange reminder for replacement of Stream Deck button ID. 

IMPORT
1. Back up Streamer.bot.
2. Import Stream_Suite_Main_Package.sb.
3. Reconnect channel-specific rewards.
4. Replace streamername and botname placeholders.
5. Review and enable only the commands your channel needs.
6. Configure private SMTP/contact values only in a private local copy.
7. Test attendance, subscriptions, resets, Stream Start, and Stream End before a production stream.
